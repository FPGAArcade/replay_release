Due to copyright reasons, ROMs are not included
and must be placed separately on the Replay SDCARD.


Lady Bug
========
Get the ROM content from a Lady Bug board and name them like this:

CPU Code EPROMs C4, D4, E4, H4, J4, K4 (4k x 8bit)
  lb1.cpu
  lb2.cpu
  lb3.cpu
  lb4.cpu
  lb5.cpu
  lb6.cpu
Character EPROMs H7, F7 (4k x 8bit)
  lb9.vid
  lb10.vid
Sprite EPROMs M7, L7 (4k x 8bit)
  lb7.cpu
  lb8.cpu
Sprite Color PROM F4
  10-1.vid
Sprite Control PROM C4
  10-3.vid
RGB Look-up PROM K1
  10-2.vid


Cosmic Avenger
==============
Get the ROM content from a Cosmic Avenger board and name them like this:

CPU Code EPROMs C4, D4, E4, H4, J4, K4 (4k x 8bit)
  1.c4
  2.d4
  3.e4
  4.h4
  5.j4
  6.k4
Character EPROMs H7, F7 (4k x 8bit)
  0.h7
  9.f7
Sprite EPROMs M7, L7 (4k x 8bit)
  8.l7
  8.l7
Sprite Color PROM F4
  10-1.f4
Sprite Control PROM C4
  10-3.c4
RGB Look-up PROM K1
  10-2.k1
