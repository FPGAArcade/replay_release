Due to copyright reasons, ROMs are not included
and must be placed separately on the Replay SDCARD.

Get the ROM content from a Dorodon board and name them like this:

CPU Code EPROMs C4, D4, E4 (8k x 8bit)
  dorodon.0
  dorodon.1
  dorodon.2
Character EPROMs H7, F7 (4k x 8bit)
  dorodon.6
  dorodon.5
Sprite EPROMs M7, L7 (4k x 8bit)
  dorodon.3
  dorodon.4
Sprite Color PROM F4
  dorodon.bp1
Sprite Control PROM C4
  dorodon.bp2
RGB Look-up PROM K1
  dorodon.bp0
